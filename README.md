# Orakle Documentation

AI-Powered Data Oracle with x402 Micropayments - Documentation Website

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프리뷰
npm run preview
```

## 기술 스택

- React 18
- TypeScript
- Vite
- React Router
- CSS Modules

## 디자인 시스템

프리미엄 수준의 미니멀 디자인으로 구현되었습니다:
- 순수 흰색 배경 (#ffffff)
- 진한 회색 텍스트 (#1a1a1a)
- 인디고 액센트 컬러 (#6366f1)
- 부드러운 애니메이션 (cubic-bezier)
- 반응형 디자인
